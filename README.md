# Hotel_Data-Scraper
*Pyhton and BeautifulSoup*
Simple Python script that extracts hotel information from a booking website and saves the results into a CSV file.

## Features
- Fetches hotel name, location, price, reviews, number of reviews, rating, and hotel link
- Uses BeautifulSoup for parsing
- Outputs a CSV data
---
## Notes
Class names and HTML structure of the target site may change. Update selectors in the script if extraction fails.
